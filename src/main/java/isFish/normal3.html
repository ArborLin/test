    中景瑞晟绩效管理与评估服务中心是我国最早成立的专业化研究公共绩效管理的民办非营利机构之一，是中国财政学会绩效管理研究专业委员会会员单位。本中心凝聚了一大批来自北京大学、清华大学、中国人民大学、中央财经大学等著名高校和科研机构的教授和学者，来自会计师事务所、大型咨询公司、全国性行业协会的绩效管理资深专家，以及来自中央部委、中央企业......

        公共绩效评价是指对政府、事业单位、国有企业等从事公共事务管理和服务的机构的政策效果、工作业绩、资金效益及其对经济社会影响等内容进行综合性考核与评价的一种活动。最近几年，在各级政府及财政部门的大力推动之下，公共绩效评价工作蓬勃发展。
